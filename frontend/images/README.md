# Novadoc Logo

## How to Add Your Logo

1. **Place your Novadoc logo file here** with the name: `novadoc-logo.png`
2. **Supported formats**: PNG (recommended), SVG, or JPG
3. **Recommended size**: 200px wide (height will auto-adjust)
4. **Transparent background** recommended for PNG files

## File Location

```
frontend/images/novadoc-logo.png
```

## If You Don't Have a Logo Yet

The application will work fine without a logo - the logo image will simply not display and the header will show just the text "Intelligent Document Search".

## To Replace the Logo

Simply replace the `novadoc-logo.png` file in this directory with your actual logo file.
